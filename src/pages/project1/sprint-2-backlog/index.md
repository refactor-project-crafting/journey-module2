# Sprint 2 Backlog

El objetivo del Sprint 2 es mejorar los estilos de la UI.

## Backlog del Sprint 2

### 1. Espacios

La caja para introducir texto tiene el texto muy pegado al borde, debe estar más separado.
Los bloques de análisis están pegados entre sí, deben tener una separación entre ellos.

### 2. Bloque frecuencias

Las listas no deben tener bullets ni espacios laterales.
El bloque "Frequencies" debe tener una disposición de tres columnas, cada palabra debe estar en una columna.

### 3. Encabezados

El título principal "Text Analyzer" debe estar subrayado.
El tamaño de letra de los encabezados de los bloques de análisis debe ser un poco más pequeño que el que tienen actualmente.

### 4. Enlaces

Al pasar el cursor por encima de un link debe desaparecer el subrayado.
El link del bloque de palíndromos debe estar más separado del texto que viene después.

### 5. Inputs

Cuando los input tengan el foco, se les tiene que poner el fondo gris oscuro y la letra blanca.
El color del borde de los inputs debe ser azul.

### 6. Esquinas redondeadas

La caja de texto debe tener las esquinas redondeadas.
Los bloques de análisis deben tener las esquinas redondeadas.

### 7. Cambiar botones

Todos los botones de la web tienen que mostrar un cursor en forma de mano cuando se pone el puntero encima.
El botón de analizar no debe ocupar todo el ancho, debe tener un tamaño fijo y estar alineado a la derecha.
El botón de analizar debe tener un borde negro de 1px de ancho.
Al pasar el cursor por encima del botón, su fondo debe ser transparente y su texto negro.
Cuando el botón esté inactivo, se debe ver con un grado alto de transparencia. Al poner el cursor encima debe cambiarse al signo de prohibido.

### 8. Cambiar colores

El fondo de la página debe tener algún color.
El color de fondo de los bloques de análisis debe ser diferente al actual.

### 9. Theme switcher

En la parte superior hay un theme switcher. Este switcher añade la clase `dark` al contenedor general de la web o se la quita, según su estado. Cuando esté activo, los colores de la web deben cambiar a un tema oscuro.
