# Módulo 2: Inmersión práctica en Desarrollo Web Full Stack

## Proyecto 2: Music Oasis

![Discos de vinilo en una tienda](/images/discos.webp)

Ahora que has demostrado que funcionas bien entrando a colaborar en un proyecto ya comenzado, es un buen momento para que te enfrentes a un proyecto completo desde cero. En este proyecto formarás parte de un equipo que tendrá que crear una aplicación web para una tienda de música llamada **Music Oasis**.

Recuerda que cada vez que vayas a crear una feature o un fix, deberás crear una rama auxiliar que siga el formato establecido en nuestras _guidelines_. No tengas la rama abierta mucho tiempo, y cuando termines tu trabajo, haz una _pull request_ para que otro miembro del equipo lo revise.
