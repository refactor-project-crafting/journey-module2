## Proyecto 5: Monuments

Una asociación cultural nos ha encargado la creación de una aplicación web para dar a conocer monumentos del mundo. La aplicación debe tener un **backoffice** tipo CRUD para que los administradores gestionen su contenido. El objetivo de este sprint es desarrollar dicho backoffice.
