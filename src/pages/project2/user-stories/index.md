# User Stories

Como usuario,

- quiero poder ver los discos de vinilo que se venden en la tienda, para poder elegir cuál comprar.
- quiero poder ver los discos digitales que se venden en la tienda, para poder elegir cuál comprar.
- quiero poder añadir discos de vinilo al carrito, para poder comprarlos.
- quiero poder añadir discos digitales al carrito, para poder comprarlos.
- quiero poder ver los discos que he añadido al carrito, para poder ver cuánto voy a gastar.
- quiero poder eliminar discos del carrito, para poder cambiar de opinión.
- quiero poder comprar los discos del carrito, para poder disfrutar de la música.
