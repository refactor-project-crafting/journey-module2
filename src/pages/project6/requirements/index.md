# Requirements

## Functional requirements

### Front

La aplicación web debe tener dos páginas, una para visualizar la lista de posts y otra para poder añadir un post.

Por ahora no implementaremos autenticación, por lo que cualquier usuario podrá añadir y eliminar posts.

### Back

Nos han proporcionado una API REST para poder interactuar con los datos. La tienes <a href="https://github.com/refactor-project-crafting/blogs-api/" target="_blank">aquí</a>.

## Technical requirements

### Front

La aplicación web debe usar React y tener los estilos con SASS. Debe comunicarse con la API REST proporcionada para obtener y enviar datos. Debe desplegarse en Netlify.

### Back

La API REST debe desplegarse en Render.
